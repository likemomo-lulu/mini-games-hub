## 📐 设计规范

### 色彩系统
- **主色调**: 根据主题动态变化
- **辅助色**: 渐变色系，用于按钮和强调元素
- **文字色**: #2D3436 (主要), #636E72 (次要), #B2BEC3 (浅色)
- **背景色**: 渐变背景，避免纯色
- **状态色**: 成功(#20B2AA), 警告(#FFD93D), 危险(#FF6B6B)

### 圆角规范
- **卡片**: 24rpx
- **按钮**: 44rpx (大), 32rpx (小)
- **游戏棋盘**: 32rpx
- **统计卡片**: 20rpx

### 间距规范
- **页面边距**: 40rpx
- **卡片间距**: 20rpx
- **元素间距**: 16rpx (小), 30rpx (中), 60rpx (大)

### 阴影系统
- **卡片**: `0 4rpx 20rpx rgba(0, 0, 0, 0.06)`
- **按钮**: `0 8rpx 24rpx rgba(0, 0, 0, 0.15)`
- **游戏棋盘**: `0 16rpx 48rpx rgba(0, 0, 0, 0.15)`

### 字体规范
- **品牌标题**: 48rpx, 800 weight
- **游戏标题**: 32rpx, 700 weight
- **统计数值**: 32-36rpx, 700 weight
- **描述文字**: 24-26rpx, regular
- **提示文字**: 26rpx, regular

## 🚀 技术特点

### 微信小程序规范遵守
- ✅ v-for 的 :key 使用简单值（不使用字符串拼接）

## 👨‍💻 开发者注意事项

### 主题开发
- 新增主题请在 `utils/theme.js` 中添加
- 主题必须包含所有必需的颜色配置
- 建议使用渐变色而非纯色

### 样式开发
- 优先使用全局样式类（避免重复代码）
- 遵循设计规范（圆角、间距、阴影）
- 使用 CSS 变量实现主题切换

### 游戏开发
- 游戏页面必须使用主题系统
- 保持统一的页面结构（header + controls + game-board + tips）
- Canvas 大小统一为 552rpx × 552rpx
